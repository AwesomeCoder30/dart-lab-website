# Design System Notes

## Design Intent

The site should borrow from the tone of the White House website:

- strong typography
- restrained palette
- clear hierarchy
- large whitespace
- minimal decorative treatment

## Palette

Use a tightly controlled palette:

- primary navy
- accent gold
- neutral white / near-black

Starter tokens for mockups:

- `navy-700: #14233c`
- `gold-500: #a57a2a`
- `white: #ffffff`
- `ink-900: #111111`
- `border-200: #d9dde3`

These are starting values for the build repo. Final production values should be confirmed against approved DART Lab branding and the live White House reference during Webflow implementation.

## Typography

- use no more than two font families total
- choose a serif + sans pairing or a strong single-family system that feels institutional
- default to high readability over novelty
- maintain strong contrast in heading hierarchy

Suggested direction:

- headings: classic serif
- body and UI: clean sans-serif

## Layout Rules

- large whitespace between sections
- consistent container widths
- avoid dense, wall-of-text layouts
- keep alignment consistent across pages
- keep section headers visually consistent

## Components

### Buttons

- one button style across the site
- filled navy primary button
- outline secondary button when needed
- avoid mixed button treatments

### Cards

- flat or lightly framed
- no gradients
- no oversized shadows
- no decorative icon clutter

### Navigation

- simple top nav
- no cluttered dropdowns
- sticky header acceptable if executed cleanly

## Motion

- avoid flashy animation
- use only minimal transitions if needed
- no parallax, autoplay video, or animated counters by default
